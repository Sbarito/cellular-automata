# SIR-
Здесь опубликована кодовая база веб сайта, который описывает SIR модель на клеточных автоматах. Данный проект взят как диплом под названием "Имитационная модель для прогнозирования пространственного и временного распространения инфекционных заболеваний". В README описана инструкция по применению и запуску
